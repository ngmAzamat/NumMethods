## Y = X * X * X - 4

Итак мы берем x = 0, вычесляем y для x = 0. y = 0 * 0 * 0 - 4 = -4.

И делаем x + 1, y = 1 * 1 * 1 - 4 = -3.

И еще пока знак не изменится: 2 * 2 * 2 - 4 = 4.

Итак берем преведущее значение отрицательное(x = 1), и положительное(x = 2), где-то тут есть наш ответ.
